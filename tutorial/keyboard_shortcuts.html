ChromeVox Keyboard Shortcuts Reference Page

  • Ctrl + Alt + Z = Start ChromeVox (must be pressed on the initial login screen)
Speech Commands
  • Control = Stop speaking the current text
  • Search = Stop speaking the current text
  • Shift + Search + Equals = Increase rate of speech
  • Shift + Search + Minus = Decrease rate of speech
  • Shift + Search + Apostrophe = Increase pitch of speech
  • Shift + Search + Semicolon = Decrease pitch of speech
  • Shift + Search + Right Bracket = Increase speech volume
  • Shift + Search + Left Bracket = Decrease speech volume

Sticky Mode

  • Double-tap Search key = Enable/Disable sticky mode

Basic Navigation

  • Shift + Search + Down = Navigate forward
  • Shift + Search + Up = Navigate backward
  • Shift + Search + Right = More detail
  • Shift + Search + Left = Less detail
  • Enter = Activate current item
  • Space = Force click on current item

Table Mode

  • Shift + Search + T + E = Enter table
  • Shift + Search + T + Backspace = Exit table
  • Shift + Search + T + M = Go to next table row
  • Shift + Search + T + I = Go to previous table row
  • Shift + Search + T + K = Go to next table column
  • Shift + Search + T + J = Go to previous table column
  • Shift + Search + T + H = Announce the headers of the current cell
  • Shift + Search + T + L = Announce the current table location
  • Shift + Search + T + R = Announce the row header of the current cell
  • Shift + Search + T + C = Announce the column header of the current cell
  • Shift + Search + T + Left Bracket = Jump to the beginning of the table
  • Shift + Search + T + Right Bracket = Jump to the end of the table

Jump Commands

Focusable items

  • Tab = Jump to next focusable item
  • Shift + Tab = Jump to previous focusable item
  • Shift + Search + N + C = Next combo box
  • Shift + Search + P + C = Previous combo box
  • Shift + Search + N + E = Next editable text area
  • Shift + Search + P + E = Previous editable text area
  • Shift + Search + N + F = Next form field
  • Shift + Search + P + F = Previous form field
  • Shift + Search + N + L = Next link
  • Shift + Search + P + L = Previous link
  • Shift + Search + N + R = Next radio button
  • Shift + Search + P + R = Previous radio button
  • Shift + Search + N + S = Next slider
  • Shift + Search + P + S = Previous slider
  • Shift + Search + N + U = Next button
  • Shift + Search + P + U = Previous button
  • Shift + Search + N + X = Next checkbox
  • Shift + Search + P + X = Previous checkbox

Headings

  • Shift + Search + N + H = Next heading
  • Shift + Search + P + H = Previous heading
  • Shift + Search + N + 1 = Next level 1 heading
  • Shift + Search + P + 1 = Previous level 1 heading
  • Shift + Search + N + 2 = Next level 2 heading
  • Shift + Search + P + 2 = Previous level 2 heading
  • Shift + Search + N + 3 = Next level 3 heading
  • Shift + Search + P + 3 = Previous level 3 heading
  • Shift + Search + N + 4 = Next level 4 heading
  • Shift + Search + P + 4 = Previous level 4 heading
  • Shift + Search + N + 5 = Next level 5 heading
  • Shift + Search + P + 5 = Previous level 5 heading
  • Shift + Search + N + 6 = Next level 6 heading
  • Shift + Search + P + 6 = Previous level 6 heading

Page Structure

  • Shift + Search + N + G = Next graphic
  • Shift + Search + P + G = Previous graphic
  • Shift + Search + N + I = Next list item
  • Shift + Search + P + I = Previous list item
  • Shift + Search + N + O = Next list
  • Shift + Search + P + O = Previous list
  • Shift + Search + N + Q = Next block quote
  • Shift + Search + P + Q = Previous block quote
  • Shift + Search + N + T = Next table
  • Shift + Search + P + T = Previous table

Getting Help

  • Shift + Search + H = Open ChromeVox User Guide
  • Shift + Search + Period = Show ChromeVox command help
  • Shift + Search = Hide ChromeVox command help
  • Shift + Search + Forward Slash = Enable/Disable search

Browser Commands

  • Shift + Search + O + K = Open keyboard explorer
